# Getting Started with AgriEdge-AI

## Prerequisites
- Android phone (mid-range OK) or Raspberry Pi Zero 2W
- Python 3.10+ for dev
- Flutter 3.x for app

## Installation
1. Clone repo
2. `cd app && flutter pub get`
3. For ML: `pip install -r requirements.txt` (in backend/)
4. Load models from Hugging Face or local.

Full guides coming in v0.1.

**Offline by design.** No internet needed after initial setup.

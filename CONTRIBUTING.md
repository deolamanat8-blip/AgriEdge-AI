# Contributing to AgriEdge-AI

Thank you for your interest! This project aims to make AI accessible to smallholder farmers worldwide.

## How to Contribute
1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Test thoroughly (especially offline)
5. Submit a PR

## Good First Issues
- Add support for a new crop/pest model
- Translate UI to new language
- Write hardware tutorial for ESP32
- Improve model quantization scripts

See issues tab for labeled `good first issue`.

Questions? Open an issue or join discussions.

Together we can feed the world smarter! 🌍
